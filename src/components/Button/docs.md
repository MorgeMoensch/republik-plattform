## Gross

Grosse Buttons sind Block-Elemente und füllen den gegebenen Platz.

```react|span-3
<Button big primary>
  Päng!
</Button>
```

```react|span-3
<Button big>
  Päng
</Button>
```

### Spezialfälle

Für Einpassungen, z.B. in einen Header, ist es zulässig eine explizite Höhe zu setzen.

```react
<Button style={{height: 50}}>
  Päng
</Button>
```
